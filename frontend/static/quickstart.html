‚ö° Quick Start

1. Start the Server

cargo run --bin avila-analytics

2. Add Tracker to Your Website

<!-- Add before closing </body> tag -->
<script src="http://localhost:8080/avila-tracker.js"
        data-site="YOUR_MEASUREMENT_ID"></script>

3. Track Events Manually (Optional)

// Track custom event
avila.track('button_click', {
    button_id: 'cta-main',
    section: 'hero'
});

// Track e-commerce
avila.trackEcommerce('purchase', {
    transaction_id: 'TXN123',
    value: 99.99,
    currency: 'USD',
    items: [{
        item_id: 'SKU001',
        item_name: 'Product',
        price: 99.99,
        quantity: 1
    }]
});
